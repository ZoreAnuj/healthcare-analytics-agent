# Healthcare Analytics Agent

A Gen AI conversational analytics platform for healthcare that replaces traditional dashboards with a natural language interface. It enables stakeholders to query complex healthcare data through intuitive conversations, transforming how insights are discovered and shared.

## Key Features
- AI-powered natural language interface for querying healthcare data
- Comprehensive stakeholder management and access controls
- Business case analysis with ROI projections
- Secure data handling with audit trails

## Tech Stack
- Python, FastAPI
- LangChain, OpenAI GPT
- PostgreSQL, Redis
- Docker, AWS

## Getting Started
```bash
git clone https://github.com/zoreanuj/healthcare-analytics-agent.git
cd healthcare-analytics-agent
pip install -r requirements.txt
python app.py
```
